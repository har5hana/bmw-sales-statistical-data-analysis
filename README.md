# BMW Sales Data Analysis (2018–2025)

This project performs statistical analysis on BMW global sales data to identify patterns, regional differences, and factors affecting vehicle sales.

The analysis uses Python for data cleaning, statistical testing, and visualization.

## Objectives

- Explore BMW global sales dataset
- Understand regional sales trends
- Perform statistical hypothesis testing
- Analyze relationships between fuel price index and vehicle sales
- Identify model popularity across regions

## Technologies Used

- Python
- Pandas
- NumPy
- SciPy
- Matplotlib
- Seaborn
- Google Colab / Jupyter Notebook

## Dataset

The dataset contains:

- Region
- Model
- Units Sold
- Fuel Price Index
- Year

Time Period: **2018 – 2025**

## Analysis Performed

### Data Exploration
- Dataset structure inspection
- Summary statistics
- Feature understanding

### Hypothesis Testing

**1. T-Test (Regional Sales Comparison)**  
Tested whether BMW sales differ significantly between Europe and Asia.

**2. Fuel Price Impact Test**  
Compared vehicle sales during high and low fuel price conditions.

**3. Chi-Square Test**

Used to examine the relationship between:

- BMW Models
- Sales Regions

## Key Insights

- Sales patterns vary significantly between regions.
- Fuel price fluctuations influence vehicle demand.
- Some BMW models are more popular in specific regions.
